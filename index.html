What is Full Stack Development?

Full stack development encompasses creating the front-end (what users interact with) as well as the back-end , which includes the server, database, and business logic. A full stack developer knows how to work with client-side technologies, such as HTML, CSS, and JavaScript, as well as with server-side languages and frameworks like Node.js, Python, and PHP, including database integrations.

  • Front-end: HTML—mark-up, CSS—styling, JavaScript—client-side logic.
  • Back-end: Server application, REST APIs, authentication, business logic.
  • Database: Relational (MySQL/Postgres) or NoSQL (MongoDB) storage.

This portfolio focuses on the front-end technologies taught in CET138.

Request flow
  1. Browser sends request (GET/POST)
  2. Server processes request and interacts with Database
  3. Server returns response (HTML/JSON)
  4. Browser renders result

HTML (HyperText Markup Language)

Just as a book cannot be written without a manuscript, a website cannot be created without HTML. It defines the structure and content of a website. HTML marks them with tags like <h1>, <p>, <a>, <img> to mark up headings, paragraphs, links, and images. It provides the basic building blocks for websites by allowing web developers to define the structure and content of each page.

Now, there are a few fundamental concepts every web developer should be familiar with:

  • Semantic tags: <header>, <footer>, <article>, <section> help organize the page content meaningfully.
  • Forms: <form>, <input>, <textarea> are used to create and collect data through forms.
  • Links & Images: <a> and <img> tags provide navigation and graphical elements.
  • Media: <video> and <audio> provide support for advanced media.
  • Accessibility: Assistive technologies require the use of descriptive alt attributes and ARIA labels.

Semantic HTML helps search engines, improves accessibility, and gives structure to content.

CSS (Cascading Style Sheets)

CSS is a tool that is used to style HTML content. It is responsible for the layout, colors, fonts, spacing, and responsiveness of a webpage.By separating the style from the content, CSS improves the look of a website and makes it easier to maintain.

Rememberable CSS concepts:

  • Selectors: select elements by class, type, id, or attributes.
  • Box Model: content, padding, border, and margin define spacing of the element.
  • Flexbox & Grid: responsive design modern layout methods.
  • Variables: consistent theming using CSS custom properties.
  • Media Queries: cater styles to various screen dimensions.

Bootstrap Framework

Bootstrap is a CSS framework that helps avoid 'reinventing the wheel' by providing ready-made components, a responsive grid, and utility classes. It not only speeds up the development process but also guarantees a consistent user interface on all devices.

Bootstrap main features:

  • Grid System: raft responsive layouts with rows and columns.
  • Components: cards, buttons, modals, navbars, and much more.
  • Utilities: spacing (m-3, p-2), text alignment, colors, and display helpers.
  • Responsive Design: fully functional on mobile, tablet, and desktops.
  • JavaScript Components: modals, dropdowns, and carousels that require little setup.

JavaScript (JS)

JavaScript is a programming language primarily used to create interactive content on web pages. It can manipulate HTML, respond to user inputs, retrieve data, validate forms, and even store data on the user's device.

Important JavaScript Features:

  • DOM Manipulation: update page content with JavaScript.
  • Events: handle user interactions such as clicks, keystrokes, and scrolling.
  • Variables & Functions: retain information and define reusable scripts.
  • LocalStorage: keep data in the browser for later use.
  • Form Validation: check that the input data is correct before sending it.
  • Vanilla JS vs Frameworks: mastering JavaScript fundamentals makes it easier to work with advanced tools such as React, Vue, and Angular.
